# Roadmap

This document tracks high-level direction. For granular work see the [issue tracker](https://github.com/ZacKienzle2/Portfolio-Optimisation/issues) and [project boards](https://github.com/ZacKienzle2/Portfolio-Optimisation/projects).

The roadmap is intentionally aspirational. Items are not commitments. Priorities shift as the project learns from users and contributors. Dates are quarters of the calendar year.

## Recently delivered (Q2 2026)

- Request-validated market-data cache (no stale snapshots) and a deduplicated data layer.
- Deterministic seeding across every Monte Carlo path (copula, historical, bootstrap, pipeline) with reproducibility tests.
- `einsum` co-moment tensors with an O(N^4) universe guard; vectorised RMT eigen-reconstruction.
- Typed `Settings` (env / TOML / defaults), structured logging, typed `MarketDataError`.
- `portfolio-opt` command-line interface.
- Cohesive, colour-blind-safe figure-style system; plotting decoupled from the optimisation layer.
- Code-derived architecture diagrams (Mermaid + Graphviz) with a CI layer-violation guard.

## Now

In active development.

- Documentation site (MkDocs Material + mkdocstrings) auto-built from docstrings and deployed to GitHub Pages, embedding the autogenerated diagrams and the worked-example notebook.
- TTL-aware caching layer with retry/backoff around market-data acquisition.
- Benchmark suite (`pytest-benchmark`) and property-based tests (`hypothesis`) for the allocation and risk primitives.
- Rewrite `main.ipynb` into a clean, narrated, parameterised end-to-end worked example with saved figures.
- Golden-master regression test over the default pipeline on the committed dataset.

## Next

Planned for the next milestone (H2 2026).

- Factor-model covariance estimation (Fama-French and statistical PCA factors) as an alternative to Ledoit-Wolf, feeding HRP/NCO/HERC.
- Backtesting engine with transaction costs, turnover constraints and walk-forward (rolling-window) validation.
- CVaR- and EVaR-driven Nested Clustered Optimisation; Riskfolio-style HERC risk-measure variants.
- Constraint-aware optimisation (sector caps, box constraints, turnover budgets) across all allocators.
- Regime-switching and mean-reverting SDE calibration beyond GBM/OU (CIR, CKLS, Heston).

## Later

Under consideration. Open issues to discuss.

- GPU / Numba acceleration for large-universe copula simulation.
- Live and alternative data adapters behind the existing repository protocol.
- Additional risk measures: Omega ratio, Relativistic VaR (RLVaR), Tail-Gini.
- Bayesian hierarchical priors for expected returns; entropy-pooling views.
- Interactive web dashboard over the pipeline outputs.
- Bridge to the sibling deep-hedging project for path-dependent allocation.

## Out of Scope

Explicitly not on the roadmap. Open an issue to challenge if you disagree.

- Live order routing, execution or brokerage integration.
- Real-money trading or portfolio custody.
- Rewrites of the numerical core outside the Python scientific stack.
